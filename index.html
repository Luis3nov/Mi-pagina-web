PARTE 1: LEVANTAMIENTO DEL SERVIDOR

Guía Rápida para Servir un HTML Local con Nginx en Windows

Para levantar un sitio web estático en localhost usando Nginx en un entorno Windows, el proceso se resume en tres pasos principales: preparación de archivos, ejecución del servidor y verificación.

1. Preparación del Entorno

El primer paso consiste en preparar tanto el servidor como el contenido.

  • Descarga y ubicación de Nginx: Se descarga la versión estable de Nginx para Windows desde su sitio web oficial. El archivo .zip se descomprime en una ubicación accesible, comúnmente en la raíz del disco como C:\nginx.
  • Creación del archivo HTML: Se crea un archivo index.html básico. Este archivo se debe colocar dentro de la carpeta C:\nginx\html, que es el directorio por defecto que Nginx utiliza para servir contenido.

2. Configuración y Ejecución

Con los archivos en su lugar, el siguiente paso es iniciar el servidor.

  • Configuración por defecto: No es necesario modificar la configuración inicial. El archivo nginx.conf (ubicado en C:\nginx\conf) ya está preconfigurado para escuchar en el puerto 80 y buscar el archivo index.html en la carpeta html.
  • Inicio del servidor: Se abre una terminal (CMD o PowerShell), se navega hasta el directorio C:\nginx y se ejecuta el comando start nginx. Esto inicia el proceso del servidor en segundo plano.

3. Verificación

Finalmente, para comprobar que el sitio está en línea, se abre un navegador web y se accede a la dirección http://localhost o http://127.0.0.1. Si todo el proceso se realizó correctamente, el navegador mostrará el contenido del archivo index.html que fue creado.

1. Memory: 4 x DIMM, Max. 128GB, 4400(O.C.)/.../2133 MHz, Un-buffered Memory*
Esto se refiere a las especificaciones de memoria de una placa base o sistema, que permite instalar hasta cuatro módulos de memoria RAM tipo DIMM, con una capacidad máxima de 128GB y soportando velocidades de hasta 4400MHz con overclocking (O.C.), y 3200MHz de forma nativa, utilizando memoria no registrada (Un-buffered).
Fuente: Dell Chile

2. Expansion slots: 1 x PCIe 4.0 x16 SafeSlots (supports x16 mode)
Esto se refiere a una ranura física PCIe x16 que admite el estándar PCIe 4.0 y utiliza 16 carriles de comunicación para lograr el máximo ancho de banda. El término "SafeSlot" indica que la ranura está reforzada con una cubierta metálica para una mejor integridad estructural, mientras que "(compatible con modo x16)" confirma que puede proporcionar todo el ancho de banda disponible para un dispositivo x16, como una tarjeta gráfica de alta gama.
Fuentes: Reddit, Crystal Group

3. Storage: Total supports 2 x M.2 slots and 6 x SATA 6Gb/s ports
Describe una configuración de almacenamiento común en las placas base modernas, que generalmente indica un total de dos ranuras para unidades de estado sólido (SSD) M.2 y seis puertos para unidades SATA tradicionales. Sin embargo, la cantidad de puertos SATA disponibles a veces puede reducirse cuando se utilizan ranuras M.2, una especificación que varía según el modelo de placa base y debe confirmarse en el manual del usuario.

Explicación de Parámetros

1. Secure Boot (Arranque Seguro)
El Secure Boot o arranque seguro, es un modo para UEFI que trae Windows desde Windows 8, y que impide la ejecución de cualquier software no firmado o certificado en el arranque del sistema. Con ello se intenta evitar la carga de malware o aplicaciones que no sean de fiar cuando inicias el ordenador, pero también afecta a sistemas operativos como las distribuciones GNU/Linux. Afortunadamente, con el paso del tiempo las distribuciones Linux se están empezando a adaptar.
Fuente: Xataka

2. Windows Boot Manager (BOOTMGR)
El Windows Boot Manager (BOOTMGR) es un pequeño software cargado desde el código de volumen de arranque. Es el responsable de ayudar a tu sistema operativo Windows (10, 8, 7 o Vista) en su proceso de arranque. El Windows Boot Manager eventualmente ejecutará el archivo winload.exe, que es el gestor de arranque utilizado por el sistema para continuar con el proceso de inicio de Windows.
Fuente: Neoguias

3. Boot Menu (Menú de Arranque)
El menú Boot en Windows es una herramienta importante que te permite seleccionar qué sistema operativo o dispositivo de arranque utilizar cuando enciendes tu computadora. Desde este menú puedes elegir arrancar desde un disco duro, una unidad de CD/DVD, un puerto USB o una unidad de red, lo que es fundamental para instalar nuevos sistemas operativos o ejecutar herramientas de diagnóstico.
Fuente: TutoWindow
